# Project Plan: Vim Learning Game

## Goal

Vimのキー操作を覚えながらゲームを進め、クリアするころにはVimの操作に習熟している状態を実現するゲームを開発する。

## Significance

- Vimは強力なテキストエディタだが、独特な操作体系のため学習コストが高い
- 既存のVimチュートリアル（vimtutor等）はドライで挫折しやすい
- ゲーム形式で楽しみながら反復練習することで、自然にキーバインドが身体に染み込む
- Vimの操作を「覚える」だけでなく「習熟する」レベルまで引き上げることが差別化ポイント

## Requirements

### 学習内容
- Vimの基本操作（移動、挿入、削除、検索、置換など）を網羅
- 段階的に難易度が上がり、ゲームクリア時にはVim操作に習熟している設計
- モーション、テキストオブジェクト、レジスタ等の中級〜上級操作もカバー

### ゲームデザイン
- ゲームスタイルは未定（複数作ってもよい）
- 楽しさと学習効果を両立するゲームメカニクス
- 反復練習が自然に組み込まれる仕組み

### 非機能要件
- キー入力に対する即座のフィードバック
- 進捗の可視化（どのキー操作をどれだけ習得したか）

## Milestones
- [ ] Phase 1: ゲームスタイルの検討・プロトタイピング
- [ ] Phase 2: コアゲームエンジンの実装
- [ ] Phase 3: Vim操作カリキュラムの設計
- [ ] Phase 4: ゲームコンテンツの作成
- [ ] Phase 5: 進捗トラッキング・習熟度判定
- [ ] Phase 6: UI/UXの完成・ポリッシュ
